-------------------
Phase Ball
-------------------

game that has challenges which you solve with math, by moving a ball into a goal along the path of an equation



Getting set up
-------------------

<p>Install Node:</p>
if you have homebrew installed, you can just:

<code>
  brew install node
</code>

else:

```
git clone git://github.com/ry/node.git
cd node
./configure
make
sudo make install
```
